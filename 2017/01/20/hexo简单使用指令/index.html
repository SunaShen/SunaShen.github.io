hexo简单使用指令

发表于 2017-01-20 | 分类于 hexo | 阅读次数

在使用hexo来写博客时老是忘记指令，给自己带来了很大的不便。
就简单的总结了一下常用的一些指令。方便自己以后使用~

创建新文章

hexo new "postName"

生成静态页面至public目录

hexo generate

开启预览访问端口（默认端口4000，’ctrl + c’关闭server）

hexo server

将.deploy目录部署到GitHub

hexo deploy

新建页面

hexo new page "pageName"

查看Hexo的版本

hexo version

查看帮助

hexo help

各种主题配置见NexT主题官网http://theme-next.iissnan.com/
